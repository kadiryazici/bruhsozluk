## **Youtube Vue3 dersleri için basit ekşi sözlük klonu**

## Gerekenler

Kurulum

> npm install | yarn

Dev server(Geliştirme sunucusu) için :

> npm run dev | yarn dev <br>

Production server (Bitmiş sunucu) için:

> npm run build | yarn build <br>
> sonra <br>
> npm run start | yarn start
